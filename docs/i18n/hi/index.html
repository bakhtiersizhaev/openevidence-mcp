Overview

OpenEvidence MCP एक unofficial MCP server है जो browser session के ज़रिए OpenEvidence से जुड़ता है। इसमें auth check, history, article-by-id और ask-with-polling सपोर्ट है।

OpenEvidence क्या है

OpenEvidence को doctors literature-backed clinical answers जल्दी देखने के लिए इस्तेमाल करते हैं। यह repository MCP bridge देती है ताकि आपकी AI tools उसी workflow में OpenEvidence को query कर सकें।
