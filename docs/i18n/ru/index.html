Что это

OpenEvidence MCP — неофициальный MCP-сервер, который работает через вашу браузерную сессию OpenEvidence. Поддерживает проверку авторизации, историю, получение статьи по ID и запуск медицинских запросов.

Что такое OpenEvidence

OpenEvidence используют врачи и медкоманды для быстрого поиска ответов по клиническим вопросам на базе доказательной медицины. Этот репозиторий даёт MCP-мост, чтобы обращаться к OpenEvidence из ваших AI-инструментов.

Инструменты

  • oe_auth_status
  • oe_history_list
  • oe_article_get
  • oe_ask

Когда это нужно врачу

  • Сравнение вариантов лечения и проверка клинических аргументов
  • Дифференциальная диагностика с продолжением по контексту
  • Быстрый разбор кейсов в Codex/Claude с историей запросов
  • Единый медицинский workflow для команды через MCP
