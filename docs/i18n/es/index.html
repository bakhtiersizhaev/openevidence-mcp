Resumen

OpenEvidence MCP es un servidor MCP no oficial que usa tu sesión de navegador. Permite validar auth, leer historial, obtener artículos por ID y ejecutar preguntas médicas.

Qué es OpenEvidence

OpenEvidence es una herramienta de evidencia clínica usada por médicos para revisar respuestas respaldadas por literatura. Este repositorio aporta el puente MCP para consultarlo desde tus herramientas de IA.

Herramientas

  • oe_auth_status
  • oe_history_list
  • oe_article_get
  • oe_ask

Cuándo lo usan médicos

  • Comparar opciones terapéuticas con contexto basado en evidencia
  • Continuar preguntas de diagnóstico diferencial con historial
  • Revisar casos clínicos en Codex/Claude sin perder contexto
  • Mantener un workflow médico común para todo el equipo
