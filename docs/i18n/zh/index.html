概览

OpenEvidence MCP 是非官方 MCP 服务器，通过你的浏览器会话连接 OpenEvidence。 支持认证检查、历史列表、按 ID 获取文章，以及提问并轮询完成状态。

什么是 OpenEvidence

OpenEvidence 常被医生用于快速查看有文献依据的临床答案。 这个仓库提供 MCP 桥接层，让你的 AI 工具能够统一调用 OpenEvidence。

工具

  • oe_auth_status
  • oe_history_list

医生何时会用到

  • 治疗方案比较与证据快速核对
  • 鉴别诊断问题的连续追问
  • 在 Codex/Claude 中做病例复盘并保留上下文
  • 团队统一临床 AI 工作流
